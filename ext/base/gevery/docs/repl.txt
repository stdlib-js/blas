
{{alias}}( N, x, strideX )
    Tests whether every element in a strided array is truthy.

    The `N` and stride parameters determine which elements in the strided array
    are accessed at runtime.

    Indexing is relative to the first index. To introduce an offset, use a typed
    array view.

    If `N <= 0`, the function returns `false`.

    Parameters
    ----------
    N: integer
        Number of indexed elements.

    x: ArrayLikeObject
        Input array.

    strideX: integer
        Stride length.

    Returns
    -------
    out: boolean
        Boolean indicating whether every element is truthy.

    Examples
    --------
    // Standard Usage:
    > var x = [ 1, 1, 1, 1 ];
    > {{alias}}( x.length, x, 1 )
    true

    // Using `N` and stride parameters:
    > x = [ 1, 0, 1, 0, 1, 0 ];
    > {{alias}}( 3, x, 2 )
    true

    // Using view offsets:
    > var x0 = new {{alias:@stdlib/array/float64}}( [ 0, 1, 1, 1, 1, 1 ] );
    > var x1 = new {{alias:@stdlib/array/float64}}( x0.buffer, x0.BYTES_PER_ELEMENT*1 );
    > {{alias}}( 5, x1, 1 )
    true


{{alias}}.ndarray( N, x, strideX, offsetX )
    Tests whether every element in a strided array is truthy using alternative
    indexing semantics.

    While typed array views mandate a view offset based on the underlying
    buffer, the offset parameter supports indexing semantics based on a starting
    index.

    Parameters
    ----------
    N: integer
        Number of indexed elements.

    x: ArrayLikeObject
        Input array.

    strideX: integer
        Stride length.

    offsetX: integer
        Starting index.

    Returns
    -------
    out: boolean
        Boolean indicating whether every element is truthy.

    Examples
    --------
    // Standard Usage:
    > var x = [ 1, 1, 1, 1 ];
    > {{alias}}.ndarray( x.length, x, 1, 0 )
    true

    // Advanced indexing:
    > x = [ 1, 0, 1, 0, 1, 0, 1, 0 ];
    > {{alias}}.ndarray( 3, x, 2, 2 )
    true
