
{{alias}}( arrays )
    Divides elements of a one-dimensional double-precision floating-point
    ndarray by the corresponding elements of a second one-dimensional double-
    precision floating-point ndarray and assigns the results to the second
    ndarray.

    The output ndarray is modified *in-place* (i.e., the output ndarray is
    *mutated*).

    Parameters
    ----------
    arrays: ArrayLikeObject<ndarray>
        Array-like object containing the following ndarrays:

        - a one-dimensional input ndarray.
        - a one-dimensional output ndarray.

    Returns
    -------
    out: ndarray
        Output ndarray.

    Examples
    --------
    > var xbuf = [ 6.0, 12.0, 20.0, 30.0 ];
    > var x = new {{alias:@stdlib/ndarray/vector/float64}}( xbuf );
    > var ybuf = [ 2.0, 3.0, 4.0, 5.0 ];
    > var y = new {{alias:@stdlib/ndarray/vector/float64}}( ybuf );
    > {{alias}}( [ x, y ] )
    <ndarray>[ 3.0, 4.0, 5.0, 6.0 ]
