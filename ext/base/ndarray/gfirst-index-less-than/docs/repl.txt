
{{alias}}( arrays )
    Returns the index of the first element in a one-dimensional ndarray which is
    less than a corresponding element in another one-dimensional ndarray.

    When comparing elements, the function checks whether an element in the first
    one-dimensional input ndarray is less than a corresponding element in the
    second one-dimensional input ndarray using the less-than operator `<`. As a
    consequence, comparisons involving `NaN` always evaluate to `false`.

    Parameters
    ----------
    arrays: ArrayLikeObject<ndarray>
        Array-like object containing the following ndarrays:

        - first one-dimensional input ndarray.
        - second one-dimensional input ndarray.

    Returns
    -------
    out: integer
        Index.

    Examples
    --------
    > var x = {{alias:@stdlib/ndarray/vector/ctor}}( [ 0.0, 0.0, 0.0, 0.0 ], 'generic' );
    > var y = {{alias:@stdlib/ndarray/vector/ctor}}( [ 0.0, 0.0, 1.0, 0.0 ], 'generic' );
    > {{alias}}( [ x, y ] )
    2
