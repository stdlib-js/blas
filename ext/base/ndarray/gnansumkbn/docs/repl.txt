
{{alias}}( arrays )
    Computes the sum of a one-dimensional ndarray, ignoring `NaN` values and
    using an improved Kahan–Babuška algorithm.

    If provided an empty ndarray, the function returns `0.0`.

    Parameters
    ----------
    arrays: ArrayLikeObject<ndarray>
        Array-like object containing a one-dimensional input ndarray.

    Returns
    -------
    out: number
        Sum.

    Examples
    --------
    > var x = {{alias:@stdlib/ndarray/vector/ctor}}( [ 1.0, -2.0, NaN, 2.0 ], 'generic' );
    > {{alias}}( [ x ] )
    1.0
