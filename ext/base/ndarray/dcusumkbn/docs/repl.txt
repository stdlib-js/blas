
{{alias}}( arrays )
    Computes the cumulative sum of a one-dimensional double-precision floating-
    point ndarray using an improved Kahan–Babuška algorithm.

    If provided an empty input ndarray, the function returns the output ndarray
    unchanged.

    Parameters
    ----------
    arrays: ArrayLikeObject<ndarray>
        Array-like object containing the following ndarrays:

        - a one-dimensional input ndarray.
        - a one-dimensional output ndarray.
        - a zero-dimensional ndarray containing the initial sum.

    Returns
    -------
    out: ndarray
        Output ndarray.

    Examples
    --------
    > var x = new {{alias:@stdlib/ndarray/vector/float64}}( [ 1.0, -2.0, 2.0 ] );
    > var y = new {{alias:@stdlib/ndarray/vector/float64}}( [ 0.0, 0.0, 0.0 ] );
    > var s = {{alias:@stdlib/ndarray/from-scalar}}( 0.0, { 'dtype': 'float64' } );
    > {{alias}}( [ x, y, s ] )
    <ndarray>[ 1.0, -1.0, 1.0 ]
