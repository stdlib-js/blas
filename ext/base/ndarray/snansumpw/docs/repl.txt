
{{alias}}( arrays )
    Computes the sum of a one-dimensional single-precision floating-point
    ndarray, ignoring `NaN` values and using pairwise summation.

    If provided an empty ndarray, the function returns `0.0`.

    Parameters
    ----------
    arrays: ArrayLikeObject<ndarray>
        Array-like object containing a one-dimensional input ndarray.

    Returns
    -------
    out: number
        Sum.

    Examples
    --------
    > var x = new {{alias:@stdlib/ndarray/vector/float32}}( [ 1.0, -2.0, NaN, 2.0 ] );
    > {{alias}}( [ x ] )
    1.0
