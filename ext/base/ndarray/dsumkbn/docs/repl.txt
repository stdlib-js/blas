
{{alias}}( arrays )
    Computes the sum of all elements in a one-dimensional double-precision
    floating-point ndarray using an improved Kahan–Babuška algorithm.

    If provided an empty ndarray, the function returns `0.0`.

    Parameters
    ----------
    arrays: ArrayLikeObject<ndarray>
        Array-like object containing a one-dimensional input ndarray.

    Returns
    -------
    out: number
        Sum.

    Examples
    --------
    > var x = new {{alias:@stdlib/ndarray/vector/float64}}( [ 1.0, -2.0, 2.0 ] );
    > {{alias}}( [ x ] )
    1.0
